## 代码段
普通代码段
```

{
    printf("hello world");
    return 0;
}
```

代码段 - 指定语言
```c

int main()
{
    printf("hello world");
    return 0;
}
```

```python
if __name__ == "__main__":
    print("hello world")
    print("pi={}".format(math.pi))
```

& (x_1, y_1) = \lambda(x_2, y_2) \newline

\begin{aligned}
